Data Collection Notice

This mod collects your Minecraft username and installed mod list when you join a server. This data is used solely for analytics and compatibility tracking.

No passwords, personal files, or server IPs are ever accessed or stored.

Installation
01

Download the mod

Grab the latest .jar from Modrinth or the official discord server's releases channel.

02

Find your game folder

In your launcher, look for a Game Directory, Open Folder, or Instance Folder button in your profile settings.

03

Add the mod

Open the mods folder inside your game directory — create it if it doesn't exist. Move the .jar file in there.

04

Launch with Forge

Select your Forge 1.8.9 profile and hit play. Don't have Forge? Most launchers have a built-in installer. If not, grab it at files.minecraftforge.net.
